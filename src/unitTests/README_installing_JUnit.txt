To use JUnit you need to add the following libraries:
1. junit-4.12.jar
2. hamcrest-core-1.3.jar

Search google for them

To Install on Eclipse:
- Navigate to Project > Properties > Java Build Path > Libraries (tab) > Import External Jar
  and add the two jar files.

To run JUnit on Eclipse:
- right click on the test java file > Run As > JUnit Test
